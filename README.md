# Entry Recorder
엔트리 작품을 동영상으로 녹화해줍니다.

## 사용법
- 아무 작품 페이지에서 **우클릭** → **녹화 시작하기**를 클릭해주세요.
- 작품이 실행되면서 녹화가 시작됩니다.
- **작품을 정지**하면 녹화 파일이 다운로드됩니다.

## 설치 방법
### 1. 확장 프로그램 다운로드
- `Code` → `Download ZIP`
- `build` 폴더만 압축 해제

### 2. 크롬 확장 프로그램 페이지 열기
chrome://extensions

### 3. 확장 프로그램 로드
- 우측 상단 **개발자 모드** 활성화
- **압축해제된 확장 프로그램 로드** 클릭
- 다운로드한 `build` 폴더 선택
